# Brute Force Detection using Splunk

This project demonstrates how to detect brute-force login attempts from SSH logs using Splunk SPL queries.

## 🔍 What’s Inside
- Sample SSH log file with failed login attempts
- SPL query to identify suspicious IPs
- Splunk alert setup guide
- Educational voice-over script
- Demo video of the detection process

## 🧠 Key Concepts
- Log parsing and field extraction
- Pattern-based detection
- Real-time alerting in Splunk
- Detection engineering fundamentals

## 🚀 Try It Yourself
1. Load the sample logs into your Splunk instance
2. Use the SPL query in `queries/spl_brute_force_detection.spl`
3. Follow `alert_setup.md` to create your own alert
